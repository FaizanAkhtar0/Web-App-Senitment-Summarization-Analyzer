Welcome to Project Data Lover

This project is designed by Khalil-UR-Rehman, Saad Ali and Faizan Akhtar for educational purposes. The goal was to create a web application that not only is able to provide sentiment analysis but also a summarization of the required text. Along with some useful plots to help understand the text.

Following are the services that this project will provide:

  • Text Sentiment Analysis
  • Text Summarization Analysis
  • Text Characteristics Plots

Sentiment Analyzer - Usage Instructions: For better results. Please provide a long text, atleast composed of 6-7 lines or 7-8 sentences. To grasp a better understanding of the emotional charateristics. Also this work ignores the emoticons analysis.

Summarization Analyzer - Usage Instructions: The Algorithm we have designed works on Extractive Technique of summarization. Also the algorithm is designed in a way that it summarizes 0.3% of the provided sentences in the text.
